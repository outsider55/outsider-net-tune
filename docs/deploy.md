# 交付与部署

## 本地运行
```bash
cd outsider-net-tune
bash install.sh
bash main.sh
```

## 快捷命令
```bash
bash install-alias.sh
x
```

## 在线一键安装（推荐）
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/outsider55/outsider-net-tune/main/bootstrap.sh)
```

安装完成后可直接运行：
```bash
x
# 或
X
```

也可以手动运行：
```bash
bash ~/.outsider-net-tune/app/main.sh
```
